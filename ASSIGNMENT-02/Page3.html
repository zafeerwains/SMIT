Attributes
The HTML attributes are used to provide more information to an HTMLelement.
  • HTML attributes only appear at start tags. it will never be on end tags.
  • All HTMLelement can have attributes
  • HTML element can have multiple attributes
  • HTML attributes usually comes in name/value pairs like name=value
  • An attributes name
  • An equal= sign
  • A value surrounded by quotation mark value

   You can also use single quotation marks depending on the situation esp.when the value contains double quotes

There are two ways to specify the URL in the href attributes

  1. Absolute URL:Links to an external websites that ia hosted on aother server.
  2. Relative URL:Links to an html page that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash,it will be relative to the current page. Example href=login.html.
