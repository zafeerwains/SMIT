Empty Element

  Empty Element are element that do not have an element content and an end tag

The <br> tag defines a line break, and is an empty element without a closing tag.

Code:

    <p> This is a <br > paragraph with a line break

Output

This is a

paragraph with a line break

A list of commonly used Empty Element.

  1. <meta />
  2. <link />
  3. <img />
  4. <br />
  5. <hr />
  6. <input />

  The best practice in HTML Empty Element is to always put a forward slash / sign before the greater than > sign.

  In this way they are closed at ther start tags. these tag are also called self closig tag.


HTML is Not Case Sensitive

  HTML Tags are not case sensitive <P> means the same as <p> <H> means the same as <h>

The HTML standard does not require lowercase tags, but W3C recommends lowercase in HTML,and demands lowercase for sticker documents types like XHTML

We will always use lowercase tag names.
